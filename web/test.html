Giới thiệu về Node Js

                   NodeJs là một nền tảng mảng mẽ được xây dựng trên Javascript được sử dụng để viết ra các Server site. Với cơ chế xử lý bất đồng bộ và non-blocking làm cho NodeJs xử lý cực nhanh và ít tốn RAM.

Logo Node Js

                    Ở bài viết này mình sẽ hướng dẫn các bạn cài đặt Node Js trên windown và chạy một ứng dụng đầu tiên bằng node js. Các bạn có thể download nodejs  trên trang chủ https://nodejs.org/en/.
